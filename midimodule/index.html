MIDI Module

Turn your iPhone or iPad into a MIDI sound module!
MIDI Module is a multitimbral MIDI module for iOS and iPadOS.

Open TestFlight

Features

- 16 MIDI channels
- Up to 64-note polyphony
- Supports General MIDI
- Supports CC#91 (Reverb) and CC#93 (Chorus)
- Background audio and AirPlay

Support for the GS, XG, and MT-32 sound maps (and others) can be added by loading custom SoundFonts with file extension .sf2 or .dls.
