Directory

This page includes links to some MIDI files and SoundFonts that can be used with NS MIDI Player.
I often use MIDI files and SoundFonts from here to test the app during development. If there's anything important that I'm missing, please let me know.

Note

Most, if not all, of these items link out to external content. Always use your best judgement when clicking on links.
If a link is broken or redirects to ads, spam, etc., please let me know ASAP.

MIDI Files

  • BitMIDI has a very large collection of MIDI files.
  • This page on midkar.com has a large selection of MIDI Karaoke files.
  • The Sample MIDI file included with NS MIDI Player is available here. It's titled "Hard Times," and (as of writing this) is in the public domain.
  • "Grabbag" from Duke Nukem 3-D made available by the original composer
  • vgmusic has some very good MIDI reproductions of video game soundtracks.
  • The Geocities MIDI Collection
  • You can also find MIDI files by googling "[song name] [artist] MIDI" which should net you good results most of the time.

Composers

  • Speedy DTM et al. have put out some very good MIDI albums for the Roland GS and Yamaha XG standards. MIDI files are linked in each video's description.
  • Anders Enger Jensen has some free MIDI files available for download on his website.

SoundFonts

General MIDI

  • FluidR3 Mono is the default SoundFont included with NS MIDI Player.
  • SGM-v2.01-YamahaGrand-Guit-Bass-v2.7.sf2 from Soundfonts4U contains great Yamaha Grand Piano samples and is overall an excellent SoundFont.
  • Roni Music hosts some good SoundFonts here.

Roland GS

  • macOS includes a fairly decent Roland GS sound bank in DLS format, located here: /System/Library/Components/CoreAudio.component/Contents/Resources/gs_instruments.dls (open Finder, choose Go > Go To Folder... in the menu bar, and paste this path in)

Yamaha XG

  • This forum post on VOGONS includes a link to download "DLS by XG," Yamaha's official XGlite sound bank (DLS format). It was provided royalty-free at some point.
    If Yamaha is still offering royalty-free licenses for this sound bank to developers, please shoot me an email!
