App Support

Need help or have feedback? Drop me a line at:

apps@nitinseshadri.com

Please include the device model, system software version, and app version you are using.

If you are having issues with specific MIDI files or SoundFonts, please include links to them or attach them to your email. You can also attach screenshots or screen recordings if desired. If adding a link, please make sure that it can be viewed by anyone with the link (paste it into a Private Browsing tab to check).
